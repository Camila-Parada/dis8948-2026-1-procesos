## Una aproximación desde la realidad virtual (virtualidad)

### Realidad extendida

- a) Realidad virtual: se ingresa en un mundo virtual utilizando herramientas como lentes o cascos. No comparte conexión con el mundo real. Se estudia el movimiento de la persona para simular perspectivas del mundo real.

- b) Realidad aumentada: es un hibrido que adquiere información del mundo real para entregar información de manera dígital para crear una experiencia. Puede utilizar un teléfono, por ejemplo. No se pueden interactuar con elementos reales.

- C) Realidad mixta: es un hubrido entre la realidad virtual y la aumentada. Se requiere de un objeto para poder interactuar con un sistema virtual sostenido por el mundo real. Se puede interactuar con todos los elementos que poseen (objetos reales, información digital, entre otros).


### Interacción a través de la presencia (Embodiment)

Este concepto se utiliza como un contexto de las experiencias que ocurren en realidad virtual.

- a) Encarnación: referente a las sensaciones de un avatar existente en un mundo virtual. Los ejemplos más comunes se centran en los juegos en primera persona, en las que el usuario interactua en el mundo real como si estuviera en el virtual. Ejemplo: "mario kart" y la sensación de conducción hace que imitemos movimientos de un auto.

- b) Corporalidad: Se centra en cómo el cuerpo percibe el mundo, siendo utilizado como una interfaz para conectar con el mundo virtual. Ejemplo: el sensor kinect.

- c) Incorporación: En el ámbito de la psicología y la ciencia cognitiva se describe como el proceso integrar una herramienta externa al esquema corporal. Ante esto, el usuario incorpora las leyes del mundo físico y el cuerpo virtual como propios. Un ejemplo es "guitar hero" y la guitarra con la cuál se puede jugar.

El acto de ingresar a un espacio virtual en una experiencia inmersiva (realidad virtual) comprende dejar el espacio real en el que la percepción se encuentra habituado (Margaret Morse). Ante este hecho se encuentra un "Espacio intermedio", un umbral que posee elementos de ambos mundos que colicionan para conectar entre si. En dicho caso el cuerpo funciona como un puente para interactuar

## Dinámica del sistema

El foco de la estética del objeto, cuya obra es un sistema de arte que se compone de reglas que gobiernan una relación.

a) Sistema estático: La obra existe como un sistema fijo. El sistema no cambia su estado interno o su apariencia en respuesta a ningún estímulo. La interacción surge a partir de la contemplación o interpretación visual del espectador.

b) Sistema dinámico-pasivo: La obra cambia y evoluciona, pero lo hace siguiendo un "reloj interno" (algo cambia según una planificación previa del artista) o respondiendo a variables ambientales (temperatura, iluminación, valor de la bolsa, etc) que no provienen directamente del actuar humano. En esta obra el visitante admira los cambios sin haber interactuado con la obra.

c) Sistema dinámico interactivo: Este sistema requiere de un usuario para que interactue. Se mantiene en estado de espera constante hasta que recibe un input humano, convirtiendose en alguien que participa como parte de la muestra. Los resultando que producen pueden ser predecibles, por lo que a largo plazo puede interpretarse como monotono.

d) Sistema dinámico-interactivo variable: un nivel más complejo y sofisticado, usualmente relacionado con la inteligancia artificial. El sistema posee un almacenamiento de información, interpretación de datos y aprendizaje, las reglas de la obra pueden cambiar según la cantidad de participantes. La obra evoluciona de forma constante a través del tiempo.
